# Report generator of Peacock

### Tips
* startup server in dev mode
`
	cd client
	yarn install
	yarn run dev
`
`
	cd server
	yarn install
	yarn run dev
`

* run static check and test
`
	cd client
	yarn run lint test
`
`
	cd server
	yarn run lint test
`
